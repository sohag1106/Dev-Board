Question-1: What are the different ways to select an element in the DOM?

We can select DOM elements in many ways such as document.getElementById, getElementsByClassName, getElementsByName, querySelector, and querySelectorAll. Using these, we can select elements and modify them as needed.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML represents the content inside an HTML element, including tags. innerText returns only the visible text inside an element, ignoring hidden elements. textContent returns all text inside an element, including hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation allows us to assign a single event listener to a parent element, which then listens for events on its child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event on a child element triggers events on its parent elements unless `event.stopPropagation()` is used.

Question-5: How do you create, add, and remove elements using JavaScript?

You can use `appendChild()` to add elements and `removeChild()` to remove elements in JavaScript.
